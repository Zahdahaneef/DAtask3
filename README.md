# 📊 Sales Data Analysis Using Python & Pandas

## 📌 Project Overview

This project demonstrates a complete sales data analysis workflow using **Python** and the **Pandas** library. The analysis includes loading a sales dataset, exploring its structure, cleaning the data, calculating key business metrics, and generating meaningful insights.

The project was completed as part of a data analysis learning task to develop practical skills in data manipulation and exploratory data analysis (EDA).

---

## 🎯 Project Objectives

* Load and read a CSV dataset using Pandas.
* Explore the dataset by checking its shape, columns, and data types.
* Clean the dataset by handling missing values and removing duplicate records.
* Calculate important business metrics such as total revenue and best-performing product.
* Generate a summary report with insights.
* Gain hands-on experience with Python for data analysis.

---

## 🛠️ Technologies Used

* Python 3
* Pandas
* OpenPyXL (for report generation)
* Jupyter Notebook / VS Code

### ✅ Day 1 – Setup & Load Data

## ▶️ How to Run the Project

1. Clone this repository.
2. Install the required library:

   ```bash
   pip install pandas openpyxl
   ```
3. Place the dataset (`sales_data.csv`) in the project folder.
4. Run the Python script:

   ```bash
   python sales_analysis.py
   ```
